Technologies used:
  • Heroku for cloud application hosting
  • MongoHQ for database persistence (Heroku add-on)
  • Papertrail for application logging (Heroku add-on)
  • Spring Data MongoDB 1.3 for data repository abstraction
  • Spring 3.2.4 for server-side MVC, DI, REST, TDD
  • AngularJS 1.2 for client-side MVC, DI, TDD
  • Bootstrap 3.0 for CSS styling, scaffolding.
  • Github for SCM
Built on Github projects:
  • heroku/devcenter-mongo-java
  • heroku/devcenter-spring-mvc-hibernate
  • angular/angular-seed
  • jjzabkar/heroku-mong-java-spring
